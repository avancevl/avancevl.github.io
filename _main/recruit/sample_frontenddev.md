## 需求規範

1. **RWD.** 桌面及手機 Responsive Web Design。
1. 必使用**JavaScript.**
1. 必使用 **React.js.**
1. 必使用 \*\*CSS Flexbox
1. 必使用**React UI framework**，從以下賽選：
1. Material UI
1. Semantic UI
1. React Bootstrap
1. React Toolbox
1. Elemental UI
1. **必使用 static page 方式（無後端）在雲端架構上架。**
1. 請用雲端架構 Heroku, Firebase, ZEIT, 或 Netilfy 上架 static page。
1. 這不是後端測驗，所以網頁和使用者互動全在前端實作。
